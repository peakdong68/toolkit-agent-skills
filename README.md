# toolkit-agents

## Quick Start

**Marketplace install:**

```
/plugin marketplace add peakdong68/toolkit-agent-skills

/plugin install kit-core@toolkit
```

**Local / development:**

```bash

git clone https://github.com/peakdong68/toolkit-agent-skills.git

/plugin marketplace add ./path/to/toolkit-agent-skills
/plugin install kit-core@toolkit

```
